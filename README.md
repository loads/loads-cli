loads-cli
==========================
CLI wrapper for loadtesting with loads


Pre-requirements
------------------------
1. > Python 3
2. virtualenv

Installation
------------------------

To install loads-cli:

~~~
$ git clone https://github.com/loads/loads-cli
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python setup.py develop
~~~


Running
------------------------

To run loads-cli:

~~~
$ loads-cli -h
~~~
